HTML5

>HTML5 (HyperText Markup Language 5) is the next major revision of HTML5 (data format designed to represent web pages). This HTML5 version is currently under development.

HTML5 specifies two syntaxes for an abstract model defined in terms of the DOM: HTML5 and XHTML5. The language also includes an application layer with numerous APIs, as well as an algorithm to handle documents with non-compliant syntax.
The work was taken over by the W3C in March 2007 after being initiated by the WHATWG. The two organizations are working in parallel on the same document in order to maintain a single version of the technology. The W3C aims to close feature additions by May 22, 2011, and finalize the specification in 2014, and encourages web developers to use HTML5 now.

CSS

The term CSS is an acronym for Cascading Style Sheets. CSS is a computer language used on the internet to style HTML or XML files. Style sheets, also called CSS files, contain code that manages the design of an HTML page.

Bootstrap

This is a collection of tools useful for creating the design (graphics, animation, and interactions with the page in the browser, etc.) of websites and web applications. It's a set that contains HTML and CSS code, forms, buttons, navigation tools, and other interactive elements, as well as optional JavaScript extensions. It's one of the most popular projects on the GitHub development management platform.
